# Готовые промпты для нового чата

Для нужной задачи **скопируй только соответствующий блок ниже** (от ` ``` ` до ` ``` ` включительно — или весь текст внутри ограждения без строк с тройными кавычками, если клиент копирует «как есть»).

Имей под рукой:

- `optimization-aurora/CONTEXT.md`
- файл ТЗ из блока (например `optimization-aurora/tasks/TASK-001-slipper-mode-core.md`)
