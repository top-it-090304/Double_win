# Материалы по проекту Honor of Aurora

Эта папка собрана для подготовки презентации по игре. В ней описаны не ассеты, а рабочая логика проекта: архитектура Godot-сцен, autoload-синглтоны, основные игровые циклы, бой, экономика базы, Корона, караваны, отряды, диалоги, персонажи и концовки.

## Файлы

- `01_architecture.md` - общая архитектура проекта, сцены, узлы, события, сохранение, переходы между локациями.
- `02_autoloads_and_core_modules.md` - главные autoload-модули и их зоны ответственности.
- `03_gameplay_mechanics.md` - бой, острова, ресурсы, здания, найм, прокачка, отдых, отряды.
- `04_crown_caravan_economy.md` - система Короны, приказы, титулы, немилость, одобрение, караваны.
- `05_dialogues_story_endings.md` - диалоговая система, сюжетные линии, развилки и концовки.
- `06_characters_and_world.md` - персонажи, их роли, взаимодействия и мир игры.
- `07_presentation_short_version.md` - короткая версия для слайдов.

## Коротко о проекте

`Honor of Aurora` / `Честь Авроры` - 2D RPG на Godot 4.4. Игрок управляет воином, исследует острова архипелага Аврора, сражается в реальном времени, возвращается на базу, развивает здания, нанимает и усиливает отряд, выполняет приказы Короны по добыче и отправке Сердцевины, а через диалоги раскрывает историю монаха-целителя, юноши Мирона, ветерана-лучника и самой Авроры.

Главный игровой ритм:

1. База: подготовка, найм, прокачка, услуги зданий, караван и диалоги.
2. Поход: телепорт/отплытие на остров, волны врагов, зачистка зон, ресурсы, боссы.
3. Возврат: сброс походных бафов, сбор шахты, тик каравана/приказа Короны, сюжетные реакции.
4. Долгосрочный прогресс: уровни героя, титулы Короны, здания, отряд, story_flags и концовки.
