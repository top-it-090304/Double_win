## О проекте

**Честь Архипелага Аврора** — изометрическое 2D‑приключение: вы исследуете острова, ведёте бой в реальном времени и развиваете связь с **отрядом** и **сюжетом**. Проект изначально ориентирован на среду **ОС «Аврора»** (сенсорный ввод, совместимый рендер) и собран как полноценная учебная/портфолио‑игра на **Godot 4.4**, а не как минимальный прототип.

Исходный код и сцены лежат в каталоге [`honor-of-aurora/`](honor-of-aurora/) — это корень проекта Godot (`project.godot`).

## Что уже есть в игре

| Направление | Содержание |
|-------------|------------|
| **Бой** | Герой‑воин: меч, щит, здоровье и уровни; союзники (в т.ч. лучник), враги, физические слои для игрока, врагов, союзников, стрел и зон |
| **Отряд** | Меню **приказов отряду**, состояние боя отряда, интеграция с геймплеем (например, подавление «ложной» атаки после закрытия UI) |
| **Сюжет и диалоги** | `DialogueManager`, реестр диалогов, окна диалогов; сюжетное состояние и пост‑финальная логика мира |
| **Мета‑прогресс** | Сохранения, сложность, баланс, менеджер геймплея, звук |
| **Корона и снабжение** | Система **немилости / одобрения Короны**, влияние на исцеление, привал, стоимости, урон лучников; **износ снаряжения** и ремонт — см. [`CHANGELOG_LORE.md`](CHANGELOG_LORE.md) |
| **Платформа** | Окно **1280×720**, растягивание `canvas_items`, эмуляция мыши с тача для UI, мобильный виртуальный ввод |

Подробные **решения по канону и лору** фиксируются в [`CHANGELOG_LORE.md`](CHANGELOG_LORE.md).

## Быстрый старт

1. Установите [**Godot 4.4**](https://godotengine.org/download) (или совместимую 4.x с тем же проектом).
2. Клонируйте репозиторий или скачайте архив.
3. В Godot: **Import** → укажите файл  
   `honor-of-aurora/project.godot`  
   либо откройте папку `honor-of-aurora` как проект.
4. Запустите сцену с **Main Scene**, заданной в проекте (меню **Project → Project Settings → Application → Run**).

> **Совет:** для целевой сборки под Аврору ориентируйтесь на экспорт с **GL Compatibility** и проверку ввода на сенсоре — в `project.godot` уже включена эмуляция мыши с касаний для стандартного UI.
