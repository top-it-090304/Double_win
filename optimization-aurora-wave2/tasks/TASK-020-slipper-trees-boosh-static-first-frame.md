# TASK-020 — SLIPPER: деревья и кусты (тайлы) без анимации, первый кадр

## Цель

В режиме **«На тапке»** (`PerformancePreset.Mode.SLIPPER`) не создавать `AnimatedSprite2D` для анимированных тайлов деревьев/кустов (слои с `tile_layer_trees_y_sort_migrate.gd`); показывать **только первый кадр** тайловой анимации (как статичный `Sprite2D`).

## Требования

- Проверка: `PerformancePreset.is_slipper_mode(SaveManager)`.
- Вне SLIPPER поведение без изменений (анимация как раньше).

## Критерии приёмки

- На «На тапке» нет проигрывания анимации мигрированных деревьев/кустов; визуал — первый кадр.
- На других пресетах визуал и анимация прежние.
